Descripton of HTML tags

ABBREVIATION ⁢abbr>
The text will be abbreviated.
EMPHASIS ⁢em>
It is a string for a highly emphasized, much more important text.
QUOTE ⁢q>
A small quotation mark will be embedded at the beginning and ending of the text.
STRONG ⁢strong>
This string defines a keyword, like product name in a review, a lead sentence in a paragraph.
MARK ⁢>
This tag highlights the text.
